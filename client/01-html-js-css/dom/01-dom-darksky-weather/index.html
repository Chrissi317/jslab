Darksky weather data testing

The script at the bottom of this page's html-code does not load any data from the darksky-server, but it uses a copied json-response for testing. Weather Info parsed:

Visit the darksky developer page to get your api-key. Then use your api-key with the following request url to see a new response.

    https://api.darksky.net/forecast/API-KEY/37.8267,-122.4233

The response should compare to the data embedded via

    <script src="testdata.js"></script>

Note the trick: The embedded script places the response in a variable and exposes the data as a script so that we can test it.
