Hello world webapp with express

First, navigate the commandline into the helloworld folder. 
      
        cd /yourpath/express-helloworld
Some helloworld-code is already copied to the index.js file. Next run: 
      
        npm init
        npm install express --save
Next start your server: 
      
        node .

And finally open the webapp: http://localhost:3000/ Also read: https://expressjs.com/de/starter/installing.html and https://expressjs.com/de/starter/hello-world.html
