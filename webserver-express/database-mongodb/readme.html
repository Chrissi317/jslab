Mongodb-connected webapp with express

Note that for this example, we need to install and run a mongodb server. See the reference Installing and starting mongodb below for reference. An example (for Mac OS) would be: 
      
        brew install mongodb
        ## homebrew might have exported the path.
        ## Then start the service via the short command and
        ## the path where mongodb should store the data
        # mongod --dbpath /yourpath/express-mongodb

        ## or if prompt is already in the webapp folder, and
        ## the database should be stored in a data-subfolder here
        mongod --dbpath ./data

        ## if the path was not exported, start with including the binary path
        # mongodb-install-directory/bin/mongod --dbpath /yourpath/express-mongodb

        # to start the mongo client
        mongo
Next, to get the example webapp running, first navigate the commandline into the express-middleware folder. 
      
        cd /yourpath/express-mongodb
Some mongo-code is already copied to the index.js file. Next run: 
      
        npm init
        npm install express --save
        npm install mongodb --save
Next start your server: 
      
        node .

And finally open the webapp: http://localhost:3000/ We can now make first POST, PUT, DELETE, or GET requests via the browsers webdeveloper tools or create websites with forms sending the requests.
