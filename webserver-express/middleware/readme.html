Middleware webapp with express

First, navigate the commandline into the express-middleware folder. 
      
        cd /yourpath/express-middleware
Some middleware-code is already copied to the index.js file. Next run: 
      
        npm init
        npm install express --save
Next start your server: 
      
        node .

And finally open the webapp: http://localhost:3000/ Also read: https://expressjs.com/de/starter/installing.html and https://expressjs.com/de/guide/writing-middleware.html and https://expressjs.com/de/guide/using-middleware.html and
